# MNIST

In this project, students are asked to classify handwritten digits using the MNIST dataset. Note that this project comes after the Neurons project. Students are encouraged to use their own neural network implementation, or explore more classical mathematical methods. Students must submit multiple different approaches to the problem, comparing their effectiveness.

## Learning outcomes

- Mathematics
